# Job Portal Backend Application

A backend REST API for a Job Portal where employers can post jobs and job seekers can search and apply. The system supports role-based access for Admin, Employer, and Job Seeker.

## Technologies Used

- Java
- Spring Boot
- Spring Security
- JWT Authentication
- Postgres
- Spring Data JPA
- Swagger / OpenAPI
- Maven

## Features

- User Registration and Login
- JWT Authentication
- Role-based Authorization (Admin, Employer, Job Seeker)
- Job Posting and Management
- Job Search
- Job Application Management
- API Documentation with Swagger
